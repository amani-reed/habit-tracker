# Habit Tracker

A personal productivity app for building consistent habits and tracking daily progress.

## Features

- Add new habits
- Edit existing habits
- Delete habits
- Prevent duplicate habits

## Built With

- React
- Vite
- JavaScript
- CSS

## Getting Started

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
